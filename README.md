# UnityChanToonShaderHDRP
UnityChanToonShader to HDRP
# 介绍

将UnityChanToonShader的功能完整的移植到了HDRP管线，并且整合了流动高光的效果。（Unity版本：2019.2.3）

流动高光效果参考了喵刀Hime的源码，在此表示十分感谢。
